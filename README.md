Fleet Management App
Fleet Management App — Android-додаток для управління автопарком, що дозволяє зберігати та керувати інформацією про автомобілі, поїздки та витрати на сервісне обслуговування.
Додаток реалізований з використанням Room для локального зберігання даних та Firebase для віддаленого доступу, синхронізації та авторизації.
Основний функціонал:
- CRUD-операції для автомобілів, поїздок та витрат (локально та віддалено)
- Room Database — локальна база даних із мінімум двома пов'язаними сутностями
- Firebase Firestore — віддалене зберігання даних
- Firebase Authentication — авторизація користувачів
- Firebase Remote Config — віддалене налаштування параметрів додатку
- RecyclerView для відображення списків
- Spinner для вибору категорій/параметрів
- Context Menu для роботи з елементами списку
- Мінімум 3 Activity та 2 Fragment для розподілу логіки
Технології
- Мова: Kotlin
- База даних: Room, Firebase Firestore
- UI-компоненти: RecyclerView, Spinner, Context Menu
- Інше: Firebase Authentication, Firebase Remote Config
